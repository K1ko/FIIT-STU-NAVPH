# The Echo Spire

**The Echo Spire** is a 2D atmospheric platformer with roguelite progression, developed in **Unity for PC as a part of Game Development Course at FIIT STU**.
You play as **Errant**, a lost soul trapped within a living tower that feeds on memories, regrets, and fragments of a broken mind.

Each ascent through the Spire reveals distorted memories, challenges your precision platforming skills, and slowly uncovers the truth behind Errantâ€™s past.

## ðŸŒ€ Core Gameplay Loop

1. Start at the base of the Spire with basic movement
2. Explore vertical levels filled with hazards and enemies
3. Defeat enemies to unlock weapons
4. Discover ethereal chests containing:
   - Memory fragments (story)
   - Permanent abilities after discovering them in chests (e.g. double jump)
5. Master movement and combat to progress upward
6. Defeat a **Guardian Boss** to unlock the next Spire layer
7. Collect memory fragments to influence the ending
